# sigma

Python validation framework(namespace package).

## Link

* [sigma.core](https://github.com/pysigma/core)
* [sigma.standard](https://github.com/pysigma/standard)

## Install

Install sigma.core and sigma.standard.

```
$ pip install sigma
```

## Dependencies

* [sigma.core](https://github.com/pysigma/core)
* [sigma.standard](https://github.com/pysigma/standard)
